# SnW Group Project

A collaborative project created by the SNW group, focusing on building a structured and functional web solution using HTML, CSS, and JavaScript.

---

##  Project Overview
This project demonstrates core web development concepts including:
- HTML page structure  
- Responsive CSS layouts  
- JavaScript DOM manipulation  
- Event handling & interactivity  
- Group collaboration using Git and GitHub  

---

##  Features
- Clean and responsive UI  
- Modular folder structure  
- Interactive components using JavaScript  
- Organized codebase for easy understanding  
- Version controlled with Git  

---

##  Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (DOM Manipulation)**
- **Git & GitHub**
